# six-axis-robot-simulation
# 六轴工业机器人运动控制仿真系统

基于 MATLAB 与 Robotics Toolbox 开发的六轴工业机器人运动学仿真 GUI 系统，支持实时姿态可视化与多模式轨迹规划。

---

## 功能特性

- **三种运动模式**
  - `Jog`（点动）：拖动滑块实时控制各关节角度，机器人姿态同步更新
  - `Trap`（梯形规划 / LSPB）：平滑加减速的梯形速度曲线轨迹规划
  - `PT`（五次多项式 / jtraj）：连续光滑的多项式插值轨迹规划

- **实时 3D 可视化**：基于 DH 参数构建六轴串联机器人模型，实时渲染三维工作空间姿态

- **轨迹跟踪图表**：动态绘制各关节位置曲线与速度曲线，直观对比规划效果

- **一键回零**：支持六轴同时回归零位，可配合规划模式执行回零轨迹

## 快速开始

```matlab
% 在 MATLAB 命令窗口中运行
fbl()
```

运行后将弹出 GUI 控制界面，左侧为三维机器人姿态视图，右侧为轨迹跟踪图表，下方为控制面板。

---

## 机器人模型

采用标准 DH 参数定义六轴串联机器人（参考 PUMA 560 构型）：

| 关节 | θ | d (m) | a (m) | α |
|------|---|--------|--------|---|
| 1 | 0 | 0 | 0 | π/2 |
| 2 | 0 | 0 | 0.4318 | 0 |
| 3 | 0 | 0.15005 | 0.0203 | -π/2 |
| 4 | 0 | 0.4318 | 0 | π/2 |
| 5 | 0 | 0 | 0 | -π/2 |
| 6 | 0 | 0 | 0 | 0 |
